Hotel Booking System — SQA Automation Project:

This project is a beginner-friendly Software Quality Assurance (SQA) automation project built using Python and pytest.
It simulates a simple hotel booking system and includes automated test cases for:

Room availability

Successful booking

Double booking prevention

Booking cancellation

Invalid room handling

Technologies Used:

Python

PyTest

PyCharm (for development)

Project Structure:
hotel_sqa_project/
│
├── app/
│   └── booking_system.py
│
└── tests/
    └── test_booking.py

How to Run the Tests

Install pytest:

pip install pytest


Run the tests:

pytest -v

Test Coverage

The following features are tested:

Check if a room is available

Book an available room

Handle already-booked rooms

Cancel an existing booking

Prevent cancellation of unbooked rooms

Validate non-existent room numbers
